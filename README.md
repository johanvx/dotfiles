# dotfiles

The configuration files are managed with [dotter].

After creating a `.dotter/local.toml` for machine-specific configuration, use `dotter deploy` to deploy.

To manually deploy, please read and follow `.dotter/{global,linux,macos,windows}.toml`.

[dotter]: https://github.com/SuperCuber/dotter
