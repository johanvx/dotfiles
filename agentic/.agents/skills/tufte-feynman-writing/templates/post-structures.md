# Post Structures (1–3 min read default)

## Structure A — Reflection with action
1. Context (who I am + constraint)
2. 3 key observations
3. Practical takeaway
4. Next steps

## Structure B — Problem → mechanism → practice
1. Problem in one paragraph
2. Underlying mechanism/principle
3. Concrete practice/checklist
4. Close with bounded claim

## Structure C — Event notes from a constrained perspective
1. Event context + guiding question
2. 3–4 selected points (claim/evidence/implication)
3. Narrow conclusion
4. Immediate actions

## Structure D — How-to explainer
1. Goal and constraints
2. Step-by-step method
3. Common failure modes
4. Quick checklist
